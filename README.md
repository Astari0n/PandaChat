# PandaChat
Программа для вывода чата твича и уведомлений с donationalerts на экран. Полезно для тех у кого нет второго монитора.
Данная программа работает поверх большинства игр в ОКОННОМ режиме, или полноэкранном в ОКНЕ.

Основные возможности:

```
- Вывод чата на экран.
- Вывод новых фолловеров в чат.
- Вывод пользователей которые вошли на стрим / вышли со стрима.
- Вывод донатов из donationalerts в чат.
- Подсвечивание выделенных сообщений (которые покупаются за баллы канала).
```

### Требования для работы

- [.NETCore Runtime](https://dotnet.microsoft.com/download)

### Сборка проекта
Для успешной компиляции требуется создать CFG.cs с подобным содержанием:
```
namespace PandaChat
{
  public class CFG
  {
    public static readonly string sVersion = "<version>";
    public static readonly string sTwitchOAuth = "<twitch oauth token>";
    public static readonly string sAppID = "<twitch appid token>";
    public static readonly string sBotName = "<twitch bot name>"; // имя аккаунта где зареган appid и oauth
  }
}
```
